# Обрезка ссылок с помощью VK API

Скрипт для сокращения ссылок через API ВКонтакте и подсчета количества кликов по сокращенным ссылкам.

## Возможности

- Сокращение длинных ссылок через сервис vk.cc
- Подсчет количества кликов по сокращенным ссылкам
- Автоматическое определение типа ссылки (обычная или уже сокращенная)

## Как установить

### 1. Получите токен VK API
1. Перейдите на [dev.vk.com](https://dev.vk.com/)
2. Создайте приложение (тип: standalone)
3. В настройках приложения скопируйте сервисный ключ доступа

### 2. Настройте окружение
Создайте файл `.env` в папке с проектом и добавьте туда ваш токен:

VK_API_TOKEN=ваш_токен_здесь

### 3. Установите зависимости
Python3 должен быть уже установлен. Затем используйте `pip` для установки зависимостей:

```bash
pip install -r requirements.txt
```
## Как использовать cокращение ссылки

``` bash
python main.py https://example.com/very-long-url
```

### Проверка количества кликов

``` bash
python main.py https://vk.cc/example
```
## Примеры работы

```bash
# Сокращение ссылки
$ python main.py https://auto.ru/cars/used/sale/vaz/vesta/1130947383-127c6e32/
https://vk.cc/cUDVSW

# Проверка кликов
$ python main.py https://vk.cc/cUDVSW
Количество кликов: 0
```

* requests
* python-dotenv
